# Voice Diary v0.1.0-rc.1

This is the first downloadable Voice Diary prerelease. Voice Diary is a local-first desktop app for recording conversations, transcribing speech, and reviewing speaker identities.

## Downloads

- Windows installer: Voice Diary Setup EXE
- macOS installer: DMG
- Linux installers: AppImage and DEB
- Checksums: SHA256SUMS.txt

## First run

The packaged app installs its Python backend runtime under the Electron user data directory on first launch. The runtime may be large and installation time depends on the selected CPU or CUDA configuration. Gated Hugging Face models also require an HF_TOKEN.

## Known limitations

- The ECAPA embedding model can fail to load on Windows because of a SpeechBrain cache path issue (WinError 123).
- PyAnnote diarization can fail when stale pyc caches remain after a transformers upgrade.
- Optional torchcodec FFmpeg DLL warnings are harmless for the in-memory audio pipeline.
- CUDA acceleration depends on a compatible NVIDIA driver; CPU fallback is supported but slower.
- macOS packages are currently unsigned and unnotarized, so Gatekeeper may require an explicit allow action.
- This prerelease has not been validated against every microphone, system-audio capture device, GPU, or Linux desktop environment.
